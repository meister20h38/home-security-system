# Raspberry Pi 5 Home Security System

Raspberry Pi 5 を活用した、既存の住居に後付け可能な「人検知機能付き防犯システム」です。 現在住んでいるアパートに防犯システムが導入されていなかったため、玄関まで行かずに訪問者を確認できる仕組みを自作しました。

## 🌟 特徴
- **人感センサーによる自動検知**: 動きを検出し、システムをトリガーします。
- **AIによる人判定**: 撮影した画像から検出モデルを用いて人間かどうかを判断し、誤検知を抑制します。
- **マルチ通知機能**:
  - 人検知時に画像をメールで室内端末へ送信します。
  - 本体のLCDにシステムの稼働状況を表示します。
- **Webベースのコントロールパネル**: Raspberry PiをWebサーバー化し、同じネットワーク内の端末（iPad等）からシステムの開始/停止操作や最新画像の確認が可能です。
- **デバイスの再利用**: 使い道のなくなった旧型iPadを、システムの管理・閲覧専用端末として有効活用しています。

## 🏗 システムアーキテクチャ
![Architecture_Diagram](./assets/システムアーキテクチャ.jpg)
1. **検知**: 人感センサーが動きを検知
2. **撮影**: USBカメラで画像を撮影
3. **判断**: 検出モデルで人間か否かを判定
4. **処理**:
   - 人間だった場合、メールを自動で送信
   - Webパネル上の画像を更新
   - LCDの状態を更新
  
## 🛠 使用技術・デバイス
ハードウェア
- **本体**: Raspberry Pi 5
- **カメラ**: USBカメラ
- **センサー**: 人感センサー（PIRセンサー）
- **ディスプレイ**: I2C対応LCD
- **操作端末**: iPad mini

ソフトウェア
- **言語**: Python3
- **Webフレームワーク**: Flask
- **通信**: smtplib（SMTPプロトコルによるメール送信）

## 🔌 配線情報

| デバイス | ピン名称 | 接続先（Raspberry Pi GPIO） |
| :----: | :----: | :----: |
| 人感センサー | vcc / gnd / data | 3.3V / GND / GPIO18 |
| LCD | vcc / gnd / scl / sda | 5V / GND / I2C SCL / I2C SDA |
